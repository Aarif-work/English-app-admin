# Word of the Day - Admin Panel

A premium, professional admin interface for managing daily English learning content and user activity.

## Features
- **Daily Word Management**: Post English words with Tamil meanings and examples.
- **Submission Monitoring**: Track all user sentence submissions in real-time.
- **User Management**: Monitor scores, ban/unban users, and manage content.
- **Score Monitoring**: Track weekly leaderboards and activity trends.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Lucide Icons
