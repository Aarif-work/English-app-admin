# One Word Wonder - Admin Panel

A premium, professional admin interface for managing daily English learning content and user activity. Built with React and structured for modern performance.

## Features
- **Publishing Studio**: Schedule and post daily English words with Tamil translations and example sentences.
- **Activity Streams**: Real-time monitoring of user sentence submissions.
- **Audience Control**: Comprehensive user management, including banning/unbanning and deletion.
- **Performance Analytics**: Weekly leaderboard tracking and engagement visualizations.
- **Secure Access**: Dedicated login interface with glassmorphism design.

## Tech Stack
- **Framework**: React 19 (Vite)
- **Styling**: Vanilla CSS (Custom tokens)
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Deployment
This project is configured for one-click deployment to GitHub Pages via GitHub Actions. Simply push your changes to the `main` branch, and the build/deploy cycle will trigger automatically.
